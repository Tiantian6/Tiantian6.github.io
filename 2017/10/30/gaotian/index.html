盒子模式

1.什么是盒子模式

盒子模式主要介绍了，盒子模型的边框,内边框和外边框以及网页山的实际应用，最后介绍如何设置圆角效果和盒子阴影效果。

2.边框

1> 有三个属性

a. color (颜色)
b. width (粗细)
c. style (样式)

2> border- width

  • border- width用来指定border的粗细程度，它的值有thin、mediumm、thick和像素值。
  • thin：设置细的边距
  • medium：默认值，设置中等的边框，一般的阅览器都将其解析为2px。
  • 像元素：表示具体的数值，自定义设置边框的宽度，如，1px、5px等，使用像素为单位设置 border粗细程度，是网页中最常用的一种方式，，border-width属性用法与bord-color一样既可以分别设置四个边框粗细，也可以同时设置四个边框的粗细。

    3> 外边距

    属性

    a.margin-left(设置左外边距)
    b.margin-top(设置上外边距)
    c.margin-right(设置右外边距)
    d.margin-botton(设置下外边距)
  • 外边距位于盒子边框外，指与其他盒子之间的距离，也就是指网页中的元素与元素之间的距离，因为body本身也就是一个盒子，也有一个外边距，这也就是body的外边距产生的。

  • 外边距与边框一样，也分为上外边距，右外边距，下外边距，左外边距，设置方式和设置顺序也基本相同。

    4> 内边距

    属性

    a.padding-left(设置左外边距)
    b.padding-right(设置右内边距)
    c.padding-top(设置上内边距)
    d.padding-bottom(设置下内边距)

  • 内边距是用来控制内容与边框之间的距离，以便更精确控制内容在盒子中的位置，内边距与外边距一样，也分为上外边距，右外边距，下外边距，左外边距，设置方式和设置顺序也基本相同。

5> border-radius的语法

  • CSS中专门针对元素的圆角效果添加了一个圆角的属性border-redius。使用这个属性就可以很轻松的为元素圆角的效果，

    语法

  • border-radius：length{1，4};

  • border-radius：length{1}，设置一个属性值表示，top-left，top-right，bottom-left四个值是一样的，也就是元素四个圆角效果是一样的。
  • border-radius：length{2}，设置两个属性值表示，top-left等于bottom-right，并且取第一个值右上角和左下角取第二个值。
  • border-radius：length{3}，设置三个属性值，第一个值是top-left和bottom-left，第三个值是，bottom-left。
  • border-radius：length{4}，设置四个属性值，第一个值是top-left，第二个值是top-right，第三个值是，bottom-right，第四个值是bottom-left。

7> 盒子阴影

box-shadow的语法
  • box-shadow：inset x-offset y-offset blur-radius color;

    box-shadow属性有五个参数每个参数都有自己的作用。

  • blur-radius:阴影模糊半径，代表阴影向外模糊的模糊范围，值越大阴影向外模糊的范围越大，阴影的边缘模糊，这个值只能是正值，如果为，0，则表示不具有模糊效果，是可选值。
  • color：阴影颜色，定义绘制阴影是所使用的颜色，阴影颜色可以十六进制颜色，RGB，RGBA，透明等，不写这个值会用阅览器的默认颜色代替，由于阅览器默认颜色不一样，因此建议不要省略这个值。
